# Demo Task: Repository Health Check

**Priority:** high
**Skill:** orient
**Tags:** [zero-resource]

## Description

Perform an initial health check of the OpenAkari-Codex repository:

1. Read the project README.md
2. List all top-level directories and verify the expected structure exists
3. Check for any open tasks in TASKS.md files
4. Verify that AGENTS.md is present and readable
5. Write a brief session log summarizing the repo state

## Done when

- A session log exists in `logs/sessions/` with repo structure summary
- No errors encountered during file reads
- Task state updated to complete
