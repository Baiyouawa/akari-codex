## Goal Description
对 `Project/AI/main.tex` 组织多 Agent 并行处理，按章节对正文进行仅文字层面的深度改写，使成稿在学术语域下更接近真实研究者写作风格。改写过程中必须严格保持原文核心含义、专业术语、论证逻辑、章节结构、LaTeX 命令与交叉引用关系不变，不改动表格、图片、公式、标签、引用键、环境结构及其他非正文要素。实施方案需包含分工、审查、回归核验与迭代机制，以尽可能降低文本中的机械化与模板化痕迹，同时确保可编译性、一致性与学术表达稳健性。

## Acceptance Criteria

- AC-1: 改写范围严格限定为正文自然语言层面，不改变文档结构与技术要素
  - Positive Tests: 章节、节、小节顺序保持一致；`\section`、`\subsection`、`\label`、`\ref`、`\cite`、公式环境、表格环境、图片环境、列表环境等 LaTeX 结构不被删除、重命名或重排；公式、表格、图片内容未被修改；仅正文句子表达发生变化。
  - Negative Tests: 改动了公式符号、表格数据、图题编号、参考文献键、章节层级；新增或删除结构环境；将大段正文移动到其他章节；重写时破坏 LaTeX 编译结构。

- AC-2: 改写后保留原意、术语与论证逻辑，不引入事实性偏差
  - Positive Tests: 关键结论、方法描述、实验设定、变量含义、术语使用与原文一致；段落间因果、递进、对比、总结关系保持可追溯；专业词汇未被误替换为不准确近义词。
  - Negative Tests: 改写后出现结论变化、条件变化、实验结果表述失真；关键术语被泛化、误译或替换；论证顺序混乱导致逻辑链断裂。

- AC-3: 文风显著去模板化，呈现更自然的人类学术写作特征
  - Positive Tests: 句式长短有变化；连接方式不重复堆叠；段落内部具有自然铺垫、转折、总结；避免连续出现同构句、空泛总述和机械列举；表达更贴近论文作者真实推演过程。
  - Negative Tests: 大量保留原有僵硬句法；全章高频重复“首先/其次/最后”“值得注意的是”等模板连接词；改写后仍呈现批量生成式节奏和过度均匀的句长分布。

- AC-4: 多 Agent 并行处理后，全文风格仍保持统一且章节间术语一致
  - Positive Tests: 同一概念在不同章节中采用统一术语；摘要、方法、实验、结论中的称谓一致；不同 Agent 产出的文段在语气、正式度、时态偏好和表述密度上经过统一。
  - Negative Tests: 同一对象出现多个不一致叫法；部分章节文风过于口语化而其他章节过于板滞；章节之间对同一方法的表述粒度冲突。

- AC-5: 全流程包含独立审查与至少一轮定向迭代修订
  - Positive Tests: 每章改写后有独立审校 Agent 或人工规则检查；发现问题后进行针对性返修；返修记录至少覆盖术语一致性、逻辑保真、LaTeX 安全、风格自然度四类检查项。
  - Negative Tests: 改写完成后直接合并，无复核；仅做表面通读，没有问题清单与返修闭环；没有针对 AI 痕迹或结构安全性的专项检查。

- AC-6: 交付物可直接回写到 `Project/AI/main.tex` 或生成等价补丁，且能稳定编译
  - Positive Tests: 提供按章节合并后的 `.tex` 文件或清晰补丁；编译通过；目录、引用、编号与原稿一致；改写痕迹可追踪。
  - Negative Tests: 交付内容无法合并；混入纯文本说明导致 LaTeX 报错；出现未闭合括号、环境错配、转义字符损坏等问题。

## Path Boundaries

### Upper Bound (Maximum Scope)
最完整方案为：先对 `Project/AI/main.tex` 做章节拆分与依赖分析，建立“章节改写 Agent + 风格统筹 Agent + LaTeX 守护 Agent + 术语一致性 Agent + 逻辑保真审校 Agent + 最终整合 Agent”的多 Agent 集群。各章节并行改写时，统一遵守共享术语表、句法改写约束、禁改对象清单与风格基线；完成后执行逐章审查、跨章术语比对、引用与标签保护检查、AI 痕迹弱化检查、段落节奏均衡化调整，以及至少两轮回归修订。最终输出整合后的 `main.tex`、改写对照说明、风险清单与复核记录。

### Lower Bound (Minimum Scope)
最简可行方案为：按一级章节拆分任务，至少由多个并行改写单元分别处理正文段落，在不改动 LaTeX 结构的前提下完成基础人类化改写；之后由一个统一审查单元进行术语一致性、结构安全性和逻辑保真检查，并完成一轮合并修订，输出可编译版本。

### Allowed Choices
- Can use: 按章节或按节拆分并行；共享术语表与禁改清单；句式重组、同义措辞替换、连接方式改写、段落内部重排但不改变论证顺序；差异比对工具；LaTeX 编译检查；人工规则审校；风格统一后处理；补丁式交付或整文件交付。
- Cannot use: 修改表格、图片、公式、数据、参考文献条目、标签键、引用键、章节结构；新增未经原文支持的观点、实验、文献、定义或结论；删除关键论证；将任务扩展为翻译、润色图表、重做排版、重构目录；输出包含破坏编译的说明性文本到 `.tex` 正文中。
