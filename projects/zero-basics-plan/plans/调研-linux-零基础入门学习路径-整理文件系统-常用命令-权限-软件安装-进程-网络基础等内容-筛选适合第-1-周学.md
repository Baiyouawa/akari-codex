## Path Boundaries

### Upper Bound (Maximum Scope)
产出一份完整的 Linux 零基础调研方案，包含：
1. 零基础学习路径总览，按“认知建立 → 基础操作 → 系统理解 → 日常管理”组织。
2. 六大核心主题的知识树与学习目标。
3. 多平台资料池，覆盖官方文档、成熟教程网站、视频课程、社区文章、中文与英文优质资料。
4. 每条资料的元数据：标题、平台、作者/机构、发布时间、链接、语言、难度、时长/篇幅、适用主题、适用阶段、推荐理由、风险说明。
5. 第 1 周学习包：按 7 天拆分学习主题、资料、练习建议与阅读顺序。
6. 去重与质量评估机制：准确性、时效性、零基础友好度、可实践性、平台稳定性。
7. Multi-Agent 执行设计：角色定义、任务输入输出、依赖顺序、冲突处理与汇总规范。
8. 最终汇总为可直接转写为 28 天 Markdown 教程的大纲草案。

### Lower Bound (Minimum Scope)
至少完成以下最简可行结果：
1. 列出 Linux 零基础入门的六大核心主题。
2. 每个主题整理至少 2 条入门资料，并记录来源。
3. 从资料中筛选出适合第 1 周学习的内容，形成清单。
4. 给出基础学习顺序与简要推荐理由。
5. 输出可供后续继续扩展为 28 天教程的结构化框架。

### Allowed Choices
- Can use: 官方文档、Linux 发行版文档、知名教程网站、公开视频课程、技术社区高质量文章、图书目录页或出版社公开样章、搜索引擎辅助检索、表格化整理、Markdown 结构化汇总、Multi-Agent 分工调研。
