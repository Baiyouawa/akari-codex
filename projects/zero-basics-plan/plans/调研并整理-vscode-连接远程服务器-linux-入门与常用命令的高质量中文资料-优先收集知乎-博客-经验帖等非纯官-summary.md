## 可直接复用的核心结论
- VSCode Remote-SSH 模块可以用“插件安装 → 密码连接 → 免密配置 → SSH config → 常见报错”作为零基础主路径
- Linux 入门模块最适合先讲路径、目录、终端，再讲权限、进程、网络
- Linux 常用命令模块应按场景教学，不应直接给新手一整页命令大全硬背
