## Findings

- Git 模块最适合采用 `Pro Git 中文版` 作为主教材，以 `git-commit` 官方手册页做命令事实核对。

- 第 2 周的 Git 进阶上界建议控制在分支、merge、冲突与远程协作；rebase 冲突可放扩展阅读。
